> 核心能力

在线代码编辑器

集成 Ace Editor，支持 C++ 语法高亮、代码补全，无需本地 IDE 即可编写代码。

实时判题反馈

提交后自动编译运行，逐用例比对输出，返回 AC / WA / TLE / RE / CE 详细结果。

题目管理后台

管理员可创建、编辑、删除题目，配置测试用例和题解，灵活管理题库内容。

> 使用流程

从选题到获得结果，仅需四步

1

选择题目

浏览题目列表，按难度（Easy / Medium / Hard）筛选，点击进入详情页查看完整描述。

2

编写代码

在集成的 Ace 编辑器中编写 C++ 代码，支持语法高亮、自动补全，提供代码模板。

3

提交判题

点击提交后，后端自动使用 g++ 编译代码，并以多组测试用例逐一运行、比对输出。

4

获取结果

即时查看判题结果：AC（通过）/ WA（答案错）/ TLE（超时）/ RE（运行时错）/ CE（编译错）。

> 平台特色

难度分级

Easy / Medium / Hard 三级难度，适合不同阶段的学习者

代码模板

每道题提供 C++ 代码框架模板，降低入门门槛

官方题解

管理员可配置参考题解，帮助学习者理解解题思路

安全认证

基于 Session + HttpOnly Cookie 的登录认证，保证账号安全

进程隔离

fork + exec 进程级隔离执行代码，CPU / 内存资源限制保护系统安全

多项控制

5 秒运行超时限制、256MB 内存上限，防止恶意代码影响服务器
